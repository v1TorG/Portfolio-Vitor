Grounded - Entretenimento

Projeto web desenvolvido com HTML e CSS que apresenta o jogo Grounded, um título de sobrevivência coop, com muitos desafios. Foi meu primeiro projeto. O site contém informações gerais sobre o mundo do jogo, sua mecânica de exploração, sobrevivência e cooperação com até três amigos, além de links para acompanhar mais sobre o jogo e acessar a página oficial e loja. Criado como prática de front-end, o projeto demonstra organização visual, conteúdo claro e foco em apresentar uma experiência envolvente para fãs do gênero de jogos de sobrevivência.

Controle de gastos

Projeto web desenvolvido com HTML, CSS e JavaScript, criado a partir da necessidade de obter maior controle financeiro. Controle de Gastos é uma aplicação web desenvolvida para ajudar o usuário a registrar e monitorar suas finanças pessoais de forma simples e prática. Com ela, é possível adicionar entradas e saídas de valores, visualizar o total de receitas, despesas e o saldo geral em tempo real.
